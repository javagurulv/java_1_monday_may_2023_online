Создать класс, который описывает анкету.
У объектов этого класса должны быть следующие свойства:
- имя человека, который отвечает на вопросы анкеты;
- два вопроса и их ответы.

* - для вопросов и ответов воспользуйтесь классом созданным
в предыдущем задании.

Цель:
- научиться создавать классы для представления
  сложных объектов из реального мира.
