Написать автоматические тесты на класс FactorialRecursion
в котором реализован метод вычисления факториала используя рекурсию:

class FactorialRecursion {

    public int factorial(int n) {
        if (n == 1){
            return 1;
        }
        return n * factorial(n - 1);
    }

}

Для этого создайте класс FactorialRecursionTest и добавьте в него
необходимые тестовые методы.

При помощи тестов найдите ошибку в коде и исправьте ее.
Формулу факториала можно найти тут:
https://ru.wikipedia.org/wiki/%D0%A4%D0%B0%D0%BA%D1%82%D0%BE%D1%80%D0%B8%D0%B0%D0%BB

Цели:
- нахождение ошибок в коде при помощи тестов;
- закрепить навык написания автоматических тестов;
- тренировка в генерации тестовых вариантов;
- повторение трёх шагов в написании автоматических тестов:
    подготовка тестовых данных,
    вызов продакшн кода,
    проверка результата.
