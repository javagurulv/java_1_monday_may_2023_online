Написать автоматические тесты на класс Calculator метод:

    boolean findAverage(int number1, int number2) {...}

Для этого добавьте в класс CalculatorTest необходимые тестовые методы.


Цели:
- закрепить навык написания автоматических тестов;
- тренировка в генерации тестовых вариантов;
- три шага в написании автоматического теста:
    подготовка тестовых данных,
    вызов продакшн кода,
    проверка результата.
