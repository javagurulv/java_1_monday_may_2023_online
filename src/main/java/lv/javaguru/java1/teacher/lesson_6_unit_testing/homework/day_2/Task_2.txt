Написать автоматические тесты на класс SquareEquation
в котором реализованы методы расчёта корней квадратного уравнения:

class SquareEquation {

    public double discriminant(int a, int b, int c) {
        return b * b - 4 * a * c;

    }

    public boolean isSquareRoot(int a, int b, int c) {
        return discriminant(a, b, c) >= 0;
    }

    public double firstRoot(int a, int b, int c) {
        double D = discriminant(a, b, c);
        return (-b - Math.sqrt(D)) / (2 * a);
    }

    public double secondRoot(int a, int b, int c) {
        double D = discriminant(a, b, c);
        return (-b + Math.sqrt(D)) / (2 * a);
    }

}

Для этого создайте класс SquareEquationTest и добавьте в него
необходимые тестовые методы.

PS: все public методы должны быть протестированы отдельно друг от друга!

Цели:
- изучить сравнение типов double при написании тестов;
- закрепить навык написания автоматических тестов;
- тренировка в генерации тестовых вариантов;
- повторение трёх шагов в написании автоматических тестов:
    подготовка тестовых данных,
    вызов продакшн кода,
    проверка результата.
