Написать автоматические тесты на класс FactorialCalculator
в котором реализован метод вычисления факториала.
Класс FactorialCalculator лежит рядом с этим файлом задания.

Для этого создайте класс FactorialCalculatorTest и добавьте в него
необходимые тестовые методы.

Цели:
- закрепить навык написания автоматических тестов;
- тренировка в генерации тестовых вариантов;
- повторение трёх шагов в написании автоматических тестов:
    подготовка тестовых данных,
    вызов продакшн кода,
    проверка результата.
