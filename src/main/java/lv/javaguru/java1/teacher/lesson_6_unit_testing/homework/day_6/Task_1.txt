Написать автоматические тесты на класс SalaryTaxCalculator
в котором реализован метод вычисления величины налога
согласно следующим правилам

- до 10к евро - налог 30%;
- от 10к до 50к евро - налог 40%;
- свыше 50к евро - налог 50%.


class SalaryTaxCalculator {

    public double calculate(double salary) {
        double tax = 0;
        if (salary < 0) {
            tax = 0;
        } else if (salary < 10000) {
            tax = (salary / 100) * 30;
        } else if ((salary >= 10000) && (salary < 50000)) {
            tax = 3000 + (((salary - 10000) / 100) * 40);
        } else if (salary >= 50000) {
            tax = 19000 + (((salary - 50000) / 100) * 50);
        } else { return tax;

        }
        return tax;
    }

}

необходимые тестовые методы.

После покрытия кода автоматическими тестами выполните рефакторинг
кода -> вынесите в отдельные методы формулы из каждого if оператора.
После выполнения рефакторинга убедитесь, что все автоматические тесты
проходят.
