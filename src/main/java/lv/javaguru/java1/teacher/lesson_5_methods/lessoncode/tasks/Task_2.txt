Написать алгоритм для определения является ли число не чётным.
Реализовать алгоритм так, чтобы им можно было легко
пользоваться повторно (многократно).

PS: подсказка - инкапсулируйте алгоритм в класс Calculator в виде метода.

class Calculator {
    boolean isOdd(int number) { ... }
}

Продемонстрируйте использование алгоритма для проверки нескольких чисел.

PS: подсказка - создайте класс CalculatorApp с main(...) методом,
создайте экземпляр класса Calculator и вызовете несколько раз
метод isOdd() и распечатайте на консоль полученные результаты.


Цели:
- взгляд на программу как на маленькую систему;
- входные параметры -> алгоритм обработки -> результат;
- реализация алгоритма в виде метода;
- научиться вызывать созданный метод.
