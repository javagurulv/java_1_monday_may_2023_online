Написать алгоритм для определения наибольшего из трёх целых чисел.
Реализовать алгоритм так, чтобы им можно было легко
пользоваться повторно (многократно).

PS: подсказка - инкапсулируйте алгоритм в класс в виде метода
и воспользуйтесь методом для нахождения максимального из двух
чисел для упрощения решения.

class MaxNumber {
    int findMax(int number1, int number2, int number3) { ... }
}

Продемонстрируйте использование алгоритма несколько раз.

PS: подсказка - создайте класс MaxNumberApp с main(...) методом,
создайте экземпляр класса MaxNumber и вызовете несколько раз
метод findMax() и распечатайте на консоль полученные результаты.


Цели:
- взгляд на программу как на маленькую систему;
- входные параметры -> алгоритм обработки -> результат;
- реализация алгоритма в виде метода.
