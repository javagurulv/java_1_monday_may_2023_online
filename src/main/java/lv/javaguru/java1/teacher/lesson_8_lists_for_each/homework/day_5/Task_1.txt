Система отзывов

Интернет магазин хочет ввести систему отзывов на своём сайте.

Идея: посетители интернет магазина могут оставлять отзывы о товаре
предлагаемом в интернет магазине.

Вам предложено разработать консольный прототип данной системы для
проверки ее функциональности перед внедрением ее на сайт
интернет магазина.

Каждый отзыв должен состоять из:
- Автор (nick)
- Текст отзыва
- Оценка (1 - 5)

Система должна позволять выполнять следующие действия:
- добавление нового отзыва;
- удаление отзыва;
- возврат всех отзывов о конкретном товаре;
- нахождение наивысшей оценки о конкретном товаре;
- нахождение наименьшей оценки о конкретном товаре;
- нахождение средней оценки о конкретном товаре.

Разработайте указанную выше систему и продемонстрируйте
корректность ее работы с помощью автоматических тестов.

Для взаимодействия с данной системой через консоль создайте
простой консольный UI (User Interface).

При разработке системы примените все этапы разработки:
- анализ требований и предметной области;
- выделение модели данных и ее представление в виде кода;
- создание бизнес логики обработки данных;
- создание UI;
- тестирование.
