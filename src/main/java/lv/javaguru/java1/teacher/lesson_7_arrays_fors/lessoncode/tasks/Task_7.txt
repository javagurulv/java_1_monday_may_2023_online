Создать программу: дневник школьника.

Задача программы заключается в управлении
оценками школьника по предметам.

Для каждой оценки программа должна хранить:
- название предмета;
- саму оценку (целое число от 0 до 10).

Программа должна позволять выполнять следующие действия:
- добавлять новую оценку;
- находить максимальную оценку по предмету;
- находить минимальную оценку по предмету;
- вычислять среднюю оценку по предмету;
- вычислять общую среднюю оценку по всем предмета.


Продемонстрировать работу программы с помощью автоматических тестов.

Написать к программе консольный UI (User Interface) для работы с ней.


Цели:
- познакомиться со структурой данных список;
- познакомиться с процессом дизайна системы;
- познакомиться с понятием UI (User Interface).
